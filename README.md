# OdasAnima

Projeto ODA: interface de organização de tarefas — arraste tarefas para a agenda e veja uma pontuação.

Arquivos principais:
- `GP-T1-EP3-Vai-dar-tempo-A.html` — página da ODA (jogo/atividade).
- `pontuacao.html` — página para visualizar pontuações salvas no browser localStorage.

Como usar localmente
1. Abra `GP-T1-EP3-Vai-dar-tempo-A.html` no navegador.
2. Clique em "Começar", arraste tarefas para o timeline e clique em "Finalizar Agenda".
3. A pontuação será salva em `localStorage` sob a chave `oda_scores`.
4. Abra `pontuacao.html` para ver pontuações salvas.
